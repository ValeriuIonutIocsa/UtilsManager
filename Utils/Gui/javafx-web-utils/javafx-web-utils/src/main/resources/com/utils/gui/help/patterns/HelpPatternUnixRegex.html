Help: Unix Regex Pattern

^ - starts with
$ - ends with
[] - matches one of the characters between brackets
. - matches any character
* - repeats the pattern before 0 to n times
+ - repeats the pattern before 1 to n times
\\ - escapes special characters
etc.
